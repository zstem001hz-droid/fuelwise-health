# FuelWise Health

A React + TypeScript frontend for tracking fitness and nutrition using the Strava API v3.

## Tech Stack

| Layer    | Technology                             |
| -------- | -------------------------------------- |
| Frontend | React 19 + TypeScript + Vite           |
| Routing  | React Router DOM                       |
| Data     | Strava API v3                          |
| Backend  | Separate repo — deployed on Render.com |
